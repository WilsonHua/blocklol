## :hammer: Development Setup

### Librairies and tools

- Vuejs(2.0) + Vuex + vue-resource
- webpack 

### Dependencies

- Node.js >= 4.0.0
- NPM >= 3.0.0

### Installation

```bash
$ npm install
```

### Running

```bash
$ # Development
$ npm run dev
$ # Build into ./build
$ npm run build
```
