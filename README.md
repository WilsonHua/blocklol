# stellarPay


A wallet over stellar ---- low fees and fast payment!

>:warning: NOTE: official release for Vue.js 2.0 not yet :construction:

## :ok: Compatibility
- v1.0 alpha (**WIP** :construction:)
    - Vue.js `2.0.3` later

## :hammer: Development Setup

    # install deps
    npm install

    # develop
    npm run develop

    # build dist files
    npm run build

    # run unit tests only
    npm run unit

    # run e2e tests only
    npm run e2e

    # lint & run all tests
    npm test
