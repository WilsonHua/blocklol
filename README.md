# stellarPay

A payment service over <a href='https://www.stellar.org'>stellar</a> ---- low fees and more fast!
More easy to use;


## :ok: Compatibility
- v1.1 alpha (**WIP** :construction:)
    - Vue.js `2.0.3` later

## :hammer: Development Setup

    # install deps
    npm install

    # develop
    npm run develop

    # build dist files
    npm run build

    # run unit tests only
    npm run unit

    # run e2e tests only
    npm run e2e

    # lint & run all tests
    npm test
